# Reaction Rush

A lightweight multi-file reaction time test website built for:
- modern aesthetics
- SEO-friendly structure
- replayability and retention
- future ad placement

## Files
- `index.html` — page structure, metadata, SEO content
- `styles.css` — full styling
- `script.js` — gameplay logic, streaks, rank system, share button
- `assets/` — placeholder folder for future images like OG/social preview graphics

## How to trial locally
1. Unzip the folder.
2. Open `index.html` in your browser.

## How to deploy
Easiest options:
- Netlify
- Vercel
- GitHub Pages

## Important before going live
Replace these placeholder values in `index.html`:
- `https://example.com/` in canonical / Open Graph URL
- `assets/og-placeholder.png` with your real social preview image
